Colores

Documentación Descripción Sintaxis
Color Se especifica el valor color p{ color: black;}
RGB Definir un color por la mezcla de los 3 colores primarios p{ color: rgb(30,30,30);}
RGBA Definir la transparencia de un color p{ color: rgba(30,30,30);}
HSL p{ color: hsl(h,s,l);} Expresa un color sRGB según sus componentes de tono
HSLA p{ color:hsla(h,s,l,a);} Podemos definir la transparencia de un color usando la función hsla

Selectores

Documentación Descripción Sintaxis
Selector Universal(*) Selecciona todos los elementos *{ declaraciones; }
Selectores de tipo Permite seleccionar elementos usando el nombre del elemento elemento{ declaraciones; }
Selector de clase Permite seleccionar elementos en función de los valores del atributo class de los elementos. Para que los elementos sean seleccionados, su atributo class debe incluir un valor que coincida con el valor del selector p class=”foo bar”> ¡HOLA MUNDO! p class=”foo baz” > ¡ADIÓS MUNDO! .foo{ declaraciones; }
Selector de ID Permite seleccionar un elemento en función del valor del atributo ID del elemento. Para que el elemento sea seleccionado, su atributo id debe coincidir exactamente con el valor dado en el selector. #foo{ declaraciones; }
Selector de Atributos Permite seleccionar elementos en función de la presencia o del valor de un determinado atributo. selector[attr] { … }
